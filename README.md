Student Information Management System (SIMS)

Overview
The Student Information Management System (SIMS) is a Digital Public Good developed as a Software Engineering final project. The system is designed to improve the management of student records by replacing manual processes with a secure and efficient digital platform.

This project aligns with United Nations Sustainable Development Goal (SDG) 4: Quality Education** by promoting better educational administration and improving access to academic information.


Problem Statement
Many educational institutions still rely on paper-based records and spreadsheets to manage student information. This results in:

- Loss of student records
- Slow retrieval of information
- Data duplication
- Human errors
- Poor communication between students and administrators

 Solution
SIMS provides a centralized platform that allows educational institutions to manage student information efficiently.

The system enables users to:

- Register students
- Manage student profiles
- Manage courses
- Record student results
- Manage attendance
- Generate reports
- Allow students to view their academic information

 Features

- Secure Login
- Student Registration
- Student Profile Management
- Course Management
- Attendance Management
- Result Management
- Report Generation
- User-Friendly Dashboard
- Responsive UI Prototype (Figma)

Software Development Methodology

This project follows the Waterfall Software Development Life Cycle (SDLC) consisting of:

1. Requirements Analysis
2. System Design
3. Prototype Development
4. Testing
5. Deployment
6. Maintenance

 System Architecture

The project uses a Three-Tier Architecture:

- Presentation Layer (User Interface)
- Application Layer (Business Logic)
- Database Layer (Data Storage)
